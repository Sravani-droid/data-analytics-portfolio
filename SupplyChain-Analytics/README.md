# Supply Chain Delay & Delivery Performance Dashboard

## Project Overview
This Tableau dashboard analyzes logistics performance across shipping modes, routes, product categories, and delivery timelines.

The goal was to identify operational bottlenecks, reduce late shipments, improve on-time delivery, and estimate penalty exposure caused by delays.

## Key KPIs
- On-Time Delivery Rate: 45.2%
- Average Delay (Late Orders): 1.6 Days
- Late Shipments: 98,977
- Penalty Exposure: $3.8M
- Worst Route: Central Asia → First Class (100% Late)

## Dashboard Insights
- First Class shipping had the highest late shipment rate.
- Certain routes consistently underperformed and created delivery risk.
- Several product categories generated higher penalty exposure.
- Delivery performance fluctuated over time, indicating process inconsistency.

## Tools Used
- Tableau Public
- Excel / CSV Data Cleaning
- Calculated Fields
- KPI Design
- Interactive Filters

## Business Impact
This dashboard can help logistics managers:
- Improve route planning
- Reduce penalties
- Increase on-time delivery rates
- Prioritize high-risk shipment lanes
