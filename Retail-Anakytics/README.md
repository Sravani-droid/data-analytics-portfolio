# Retail Profitability Dashboard

## Overview

This Tableau dashboard analyzes retail sales and profitability performance across categories, products, and regions to identify revenue drivers, margin leakage, and discounting impact.

## Objective

To help stakeholders understand:

* Which product categories drive profit vs loss
* How discounting impacts profitability
* Which regions generate the highest margins
* Which products are top and bottom performers

## Dataset

* **Source:** Tableau Sample Superstore Dataset
* **Records:** ~10,000+ Orders
* **Tool Used:** Tableau Public

## Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Profit Margin KPI
* Sales by Category
* Profit by Sub-Category
* Regional Profitability Map
* Discount vs Profit Analysis
* Top / Bottom Performing Products
* Interactive Filters for Region / Category / Segment

## Key Insights

* **Technology** consistently delivers the strongest profit margins.
* **Furniture** generates high sales volume but significantly lower profitability.
* Heavy discounting strongly correlates with reduced profit margins.
* Certain sub-categories contribute sales while remaining unprofitable.
