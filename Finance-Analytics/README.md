# Banking Transaction Risk & Customer Behavior Dashboard

## Overview

This Tableau dashboard analyzes banking transaction patterns, customer demographics, and high-value transaction behavior to uncover insights into customer financial activity and potential risk indicators.

## Objective

To help stakeholders understand:

* Transaction trends over time
* Customer segment transaction behavior
* High-value transaction concentration by location
* Spending behavior across balance groups

## Dataset

* **Source:** Banking Transactions Dataset
* **Records:** 1M+ Transactions
* **Tool Used:** Tableau Public

## Dashboard Features

* Monthly Transaction Value Trend
* Monthly Transaction Count Trend
* KPI Cards:

  * Total Transaction Value
  * Total Transactions
  * Average Transaction Value
  * High Value Transactions
* Transaction Value by Gender
* Transaction Value by Age Group
* High Value Transactions by Location
* Average Transaction by Balance Group
* Interactive Filters:

  * Month
  * Gender
  * Balance Group
  * High Value Flag

## Key Insights

* Customers aged **35–44** contribute the highest transaction value.
* **Very High Balance** customers have the highest average transaction size.
* Major urban locations generate the most high-value transactions.
* Transaction activity shows an increasing trend across the observed months.
